# Odoo Migration Guides

A comprehensive collection of migration guides for upgrading Odoo modules across different versions, focusing on backend changes, ORM updates, XML views, and more.

## Guides Included

- **[Odoo Migration Guide (16 → 17 → 18 → 19)](odoo_migration_guide.md)**: A detailed overview of backend overhauls, including ORM, views, and controllers. Covers major breaking changes like the removal of `attrs`, `name_get`, and more.

- **[Odoo 17 to 18 Migration Guide](odoo_18_migration_guide.md)**: Specific changes for migrating from Odoo 17 to 18, including XML view updates (`tree` to `list`), removal of `attrs` and `states`, Python framework changes, and ORM command objects.

- **[Odoo 18 to 19 Migration Guide](odoo_19_migration_guide.md)**: Upcoming changes for migrating to Odoo 19, covering Python environment requirements, ORM and database refactoring, frontend OWL updates, and security enhancements.

## Key Topics Covered

- XML View Migrations (attrs, states, tree/list tags)
- Python ORM Changes (name_get, states, decorators)
- Frontend and OWL Updates (Kanban, templates, routing)
- Security and Configuration Updates
- Best Practices for Smooth Upgrades
